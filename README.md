# Linear Regression from Scratch

## Overview

This project implements Multiple Linear Regression from scratch using Numpy.

The objective is to understand the mathematics behind Linear Regression without relying on scikit-learn.

## Features

- Multiple Linear Regression
- Gradient Descent
- Z-score Normalization
- Prediction on new samples
- Cost vs Iterations visualization
- Actual Target value vs Predicted value visualization

## Dataset

Synthetic Student Performance dataset.

Features:
 - Hours Studied
 - Sleep Hours
 - Attendance
 - Previous Scores

Target:
 - Final Exam Score

## Technology
- Python
- Numpy
- Pandas
- Matplotlib

## My Learning Outcomes

During this project I learned:
- How gradient descent updates parameters
- Why feature scaling is important
- How Z-score normalization works
- How Linear Regression minimizes its goal (cost function)
- How to make predictions using learned parameters
